# postgrest_py

PostgREST client for Python. This library provides an ORM interface to PostgREST.

## INSTALLATION

### Requirements

- Python >= 3.7
- PostgreSQL >= 12
- PostgREST >= 7

### Instructions

#### With Poetry

```sh
$ poetry add git+https://github.com/lqmanh/postgrest_py.git#v0.1.1
```
